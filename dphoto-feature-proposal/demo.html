dphoto Feature Demo

Interactive demonstration of proposed enhancements

Photo Lightbox

Click any photo to open the enhanced lightbox viewer with swipe gestures, keyboard navigation, and action buttons.

Social Sharing

One-tap sharing with native share sheet on mobile, or clipboard copy on desktop.

Save to Photos

Download full-resolution images with meaningful filenames in one click.

Technical Features

Built with modern web standards for the best user experience.

Mobile-First

Uses Web Share API for native share sheets on iOS and Android. Touch gestures for swipe navigation.

Desktop Support

Keyboard navigation with arrow keys and Escape. Graceful fallbacks for all features.

Fast Loading

Lazy loading images, CSS animations, and minimal JavaScript for optimal performance.

Accessibility

ARIA labels, keyboard navigation, and proper focus management for all users.

Easy Integration

Modular design with CSS variables for easy customization and branding.

No Dependencies

Pure JavaScript with no external libraries required. Works with any framework.

Integration Example

Simple code snippet showing how these features could be added to dphoto.

// Initialize the photo modules
const shareManager = new PhotoShareManager({
    photographer: 'Fred Assaf',
    hashtags: ['paceacademy', 'photography']
});

const downloadManager = new PhotoDownloadManager({
    photographer: 'fredassaf',
    showConfirmation: true
});

const lightbox = new PhotoLightbox({
    photos: albumPhotos,
